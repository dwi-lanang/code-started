1. Start xampp (Webserver)

  • Install xampp.
  • Default posisi folder xampp berada di drive C:\xampp.
  • Aktifkan controll xampp
  • Buka browser kemudian ketik pada url http://localhost/xampp

2. Step 2 (Create Project)

  • Buat folder didalam folder htdocs dengan nama project kamu contohnya `webku`.
  • Buat file index.php pada folder project
  • Buka file index.php, ketik perintah 
    <?php echo "Hello World...!!!"; ?>
  • Buka browser kemudian ketik pada url http://localhost/webku/index.php

3. Step 3 (Database Connection)

  • tutorial dasar mysql
  • Buka browser kemudian ketik url http://localhost/phpmyadmin
  • Buat database baru kemudian berikan nama, contohnya `belajar`
  • Buat file baru dan beri nama `koneksi.php`
  • Buka file koneksi.php, ketik perintah 
    									<?php
										$conn = mysql_connect("localhost", "root", "");
										mysql_select_db( "belajar" , $conn);
										if (!$conn) {
										   echo "Tidak dapat konek ke DB: " . mysql_error();
										   exit;
										}
										if (!mysql_select_db("belajar")) {
										   echo "Tidak dapat memilih database belajar : " . mysql_error();
										   exit;
										}
										else
										{
											echo "Koneksi database belajar berhasil.";
											exit;
										}
									?>
  • Buka browser kemudian ketik pada url http://localhost/webku/koneksi.php
